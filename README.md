# Tercera Entrega Django

## Cómo ejecutar el proyecto
1. Activar entorno virtual
2. Ingresar a la carpeta proyecto
3. Ejecutar `python manage.py migrate`
4. Ejecutar `python manage.py runserver`
5. Abrir `http://127.0.0.1:8000/`
6. Para el panel admin abrir `http://127.0.0.1:8000/admin/`

## Funcionalidades
- Página de inicio con herencia de templates
- Modelos: Cliente, Producto y Pedido
- Panel de administración habilitado
